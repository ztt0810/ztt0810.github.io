Git常用操作

查看当前分支：

git branch

查看远程分支：

git branch -a

更新本地分支：

git remote update origin --prune
git fetch origin --prune    git branch -a

切换本地分支（若分支在本地未与orgin同步）：

git checkout -b [branch name]

删除远程分支：

git push origin --delete [branch name]

删除本地分支（dev分支）：

git branch -D dev

拉取代码（拉取指定分支）：

git pull <远程主机名> <远程分支名>:<本地分支名>

拉取代码（与当前分支合并）:

git pull <远程主机名> <远程分支名>

手动合并（打开merge工具） ：

git merge tool 
# 打开beyondcompare后左边是当前分支代码，中间是修改前的代码，右边是被合并分支的代码，点击左边箭头进行选择并合并

删除远端文件（保留本地文件）:

文件: git rm –cache filename

文件夹: git rm -r –cache dirname

创建和删除标签：

创建：git tag <标签名>

git push origin <标签名>

删除：git tag -d <标签名>

git push origin :refs/tags/标签名
